# Alpha Contable 📊
> Desarrollo de software contable optimizado con Angular 20.

Este proyecto forma parte del ecosistema **OSZ Software** (One Secure Zone) y está diseñado para gestionar procesos contables, clientes y cálculos financieros de manera eficiente.

## 🚀 Estado del Proyecto
- **Frontend:** Angular 20.3.6 + Tailwind CSS.
- **Backend:** (En planeación: Node.js + SQLite).
- **Despliegue:** cPanel (Subdominio).

## 🛠️ Instalación y Configuración
Para correr este proyecto localmente:

1. Clonar el repositorio:
   ```bash
   git clone [https://github.com/Edyckzon/alpha-contable.git](https://github.com/Edyckzon/alpha-contable.git)

Instalar dependencias:

Bash
npm install
Servir la aplicación:

Bash
ng serve
🏗️ Estructura de Componentes
calculator: Lógica para cálculos financieros.

clients: Gestión de base de datos de clientes.

navbar/footer: Componentes globales de navegación.
